# ESP32 Data Uplink Project

This branch contains a PlatformIO project for the ESP32 microcontroller.  
It is responsible for receiving data from an STM32 device and uploading it to the cloud.

## Overview

- **Platform**: ESP32 (via PlatformIO)
- **Function**: Receive sensor or control data from STM32
- **Communication**: UART
- **Cloud Service**: Firebase
## Purpose

This project is part of a system where the STM32 collects or generates data,  
and the ESP32 acts as a bridge to forward that data to an online cloud service for storage, monitoring, or processing.
